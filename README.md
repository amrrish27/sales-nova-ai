# 🌟 Overview

**Sellora AI** is an autonomous AI sales employee designed to help businesses automate sales conversations and improve conversion rates.

Instead of hiring large sales teams, Sellora AI can:

- Qualify leads automatically
- Handle customer objections
- Recommend products intelligently
- Predict close probability
- Analyze customer psychology
- Improve conversion performance

The system works through an intelligent **AI Sales Chat**, analytics dashboard, CRM integration, and a real-time **Sales Brain** engine.

# 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/amrrish27/sales-nova-ai.git
```

Go into the project folder:

```bash
cd sales-nova-ai
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

The app will run locally at:

```bash
http://localhost:5173
```

---

# 🔧 Environment Variables

Create a `.env` file in the root folder.

Example:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_key
GEMINI_API_KEY=your_gemini_api_key
```
